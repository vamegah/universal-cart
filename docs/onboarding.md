# Developer Onboarding – Universal Cart

## Prerequisites

- Node.js 18+ and npm/pnpm
- Docker and Docker Compose
- PostgreSQL client (psql) optional
- Python 3.10+ (for matching engine)
- Flutter SDK (for mobile development)
- Playwright (for e2e tests)

## Clone & Install

```bash
git clone https://github.com/universal-cart/universal-cart.git
cd universal-cart
pnpm install   # or npm install (workspaces)
